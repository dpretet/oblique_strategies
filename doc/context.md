# Oblique Strategies App - Context

## Concept
**Oblique Strategies** are a creative tool developed by Brian Eno and Peter Schmidt. They consist of a deck of cards with enigmatic phrases or instructions designed to stimulate creativity, break artistic blocks, and encourage unexpected approaches.

## Application Goal
Create a Python3 application that allows users to:
- **Randomly draw** an oblique strategy.
- **Display** the strategy in an aesthetic and interactive way.
- **Customize** strategies (add, remove, modify).

## Target Audience
- Artists, musicians, writers, or anyone seeking to boost creativity.
- Developers interested in contributing or extending the application.

## Technologies
- **Python3**: Primary language.
- **Rich**: Library for rich and interactive terminal display.
- **JSON**: Format for storing strategies, enabling easy customization.

## Key Features
1. **Random Draw**: Select a strategy at random.
2. **User Interface**: Use Rich for colored and structured display.
3. **Interactivity**: Allow users to draw a new strategy or exit.
4. **Strategy Management**: Load and save strategies from a JSON file.
5. **Customization**: Add, modify, or delete strategies.

## Example Usage
```bash
python main.py
```
Displays a random oblique strategy with an attractive design using Rich.
